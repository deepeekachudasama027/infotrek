CLUEMINATI

Guess if you can!

Event Description:

Show the Detective within you, fasten your seat belt and get ready for a 4-day long fun event. Remember the Chinese proverb: "One picture is worth ten thousand words". So use your clever mind in finding a hidden message in every image. Collect as many coins as you can and be a Numismatist.

Rules:

1) The image itself will tell you a lot about the answer, you can take the help of the internet. If you still can’t figure out the answer you can go for the clues.
2) Every player will carry 1000 coins at the beginning of the game.
3) There will be 3 clues for each challenge, unlocking the 1st clue will cost you 20 coins, 2nd clue will cost 30 coins, and 3rd will cost 50 coins.

LETTER BOX

Let's go Old School

Event Description:

Is it your fantasy to post a letter to one of your friends from your batch? Aren't you tired of sending messages and stickers on WhatsApp? Want to send a funny one anonymously? We are here with a letterbox service where you can send a letter that we will drop to the receiver's mailbox anonymously.

Rules:

1) Your letter should not be offensive to anyone.
2) No use of abusive language.
3) You can send atmost five letters.
4) Two winners will be chosen. One whose letters are least recognized by the receiver and one who writes the funniest/creative letter.
5) Remember that if you get letter in your mailbox then you've to reply to it.

CODE-ORB

EKLAVYA of Coding World!

Event Description:

COD-ORB is an exciting event where you need to solve 3 questions within 30 mins. Not only you have to break the logic of the problem but you've to write the code blindfolded i.e. you won't see the code. The participant has to solve the question correctly and his/her code should compile and run without any errors. The one who solves most of the questions correctly in the least time will win. The top 3 contestants of the event will be awarded.

Rules:

3 questions will be provided to each participant and he/she has to write the correct code to solve the problems given to them. The catch in this event is, you won't be able to see the code while you write it. The one who submits the correct code within the time limit i.e., 30 minutes will be the winner. You can submit your code once only.

CODIGO

Use what you have learnt!

Event Description:

“Practice makes a man Perfect“. Competitive coding is a way to improve your coding skills. Compete with your peers in this 2 hour long coding challenge. The candidate with highest number of points will win. Any kind of plagiarism will lead you to get disqualified from all the events.

PLATFORM: Codechef

Rules:

Once you start the event, you won't be able to change the tabs. You won't be allowed to copy/paste the code. Your cameras will be turned on for complete 120 mins until you finish the event. If any of these rules are manipulated, the participant won't be considered for any of the events.

MEME BLAST

Let's laugh together

Fight till your last breath!

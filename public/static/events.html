CLUEMINATI

Event Description:
Show the Detective within you, fasten your seat belt and get ready for a 4-day long fun event. Remember the Chinese proverb: one picture is worth ten thousand words, use your clever mind. Find a message hidden in the image, collect as many coins as you can and be a numismatist.

Rules:
1) The image itself will tell you a lot about the answer, you can take the help of the internet. If you still can’t figure out the answer you can go for the clues.
2) Every player will carry 1000 coins at the beginning of the game.
3) There will be 3 clues for each level, unlocking the 1st clue will cost you 20 coins, for the 2nd clue it will cost you 30 coins and for the 3rd it will be 50 coins.

SURPRISE EVENT

Event Description:
Be a champion of computer science. Compete in this quiz game and show that you deserve to be called a Champion among your peers. This quiz game will have 10 questions (MCQ) and 2 life-lines. Three top participants will get a chance to play this quiz. This game will be played in 2 stages. The quiz will be a rapid fire.

Rules:
The quiz will contain 10 questions. These questions will be based on topics like Problem- Solving, Data Structure, Algorithms, OOPS, Operating Systems, DBMS, etc. and the difficulty level will increase with every subsequent question. Each right answer will give you some points and the candidate with highest score will be the winner. Candidates will get 2 lifelines:-
1. “extra chance” - player will get one extra chance to answer.
2. "call your friend " - player can ask the answer to one of his friends but the time limit will be only 30 seconds. The quiz will have 2 save points.
3. If you give an incorrect answer before the next safe-point you will only get a point that was earned before the last safe-point.

LETTER BOX

Event Description:
Is it your fantasy to post a letter to one of your friends from your batch? Aren't you tired of sending messages and stickers on WhatsApp? Want to send a funny one anonymously? We are here with a letterbox service where you can send a letter that we will drop to the receiver's mailbox. The one that will have the highest number of letters in the box is the popular one. You've to recognize him/her otherwise you'll lose points.

Rules:
1) Your letter should not be offensive to anyone.
2) No use of abusive language.
3) You can only send one letter.
4) Two winners will be chosen. One who recognizes the maximum number of letters correctly and one who writes the funniest/creative letter.

CODE-ORB

Event Description:
COD-ORB is an exciting event where you need to solve 3 questions within 30 mins. Not only you have to break the logic of the problem but you've to write the code blindfolded i.e. you won't see the code. The participant has to solve the question correctly and his/her code should compile and run without any errors. The one who solves most of the questions correctly in the least time will win. The top 3 contestants of the event will be awarded.

Rules:
3 questions will be provided to each participant and he/she has to write the correct code to solve the problems given to them. The catch in this event is, you won't be able to see the code while you write it. The one who submits the correct code within the time limit i.e., 30 minutes will be the winner. You can submit your code once only.

CODIGO

Event Description:
“Practice makes a man Perfect “. Competitive coding is a way to improve your coding skills. Compete with your peers in this 2 hour long coding challenge. Their will be 4 problems, each will carry some points according to the difficulty level. The candidate with highest number of points will win. Any kind of plagiarism will lead you to get disqualified from all the events.

Rules:
Once you start the event, you won't be able to change the tabs. You won't be allowed to copy/paste the code. Your cameras will be turned on for complete 120 mins until you finish the event. If any of these rules are manipulated, the participant won't be considered for any of the events.

CODE-CONQUERORS

Event Description:
Participate in a 2-days long coding marathon. Use all the knowledge you have, to come with the most optimized solution for the given problems. You will be given sets of problems each day. Each problem will give you some points depending on the difficulty level. The one with the most points will be the winner.

Rules:
1. You won't be allowed to copy/paste the code.
2. Any kind of plagiarism will lead you to get disqualified from all the events.
3. If any of these rules are manipulated, the participant won't be considered and won't be allowed in any of the events.
